2. Github:

- create a new repository on the command line:
{
    ${
        git init
        git add README.md
        git commit -m "first commit"
        git branch -M main
        git remote add origin https://github.com/codevcnn/Lib.git
        git push -u origin main
    }
}

3. Git:

- push an existing repository from the command line:
{
    ${
        git remote add origin https://github.com/codevcnn/Lib.git
        git branch -M main
        git push -u origin main
    }
}

- merge code from branch A to branch B (branch B is gonna be overwritted):
{
    # firstly, commit all changes from the working branch, then switch to the branch want to merge to (the branch...
    # is gonna be overwritted, this case is B), then run `git merge A`, after all the branch B now has all changes...
    # of branch A. OK done!
    VD:{
        # Switch to a branch
        git checkout mybranch
        # Edit some files
        git add <file>
        git commit -m "do something"
        # Edit some files again
        git add <file>
        git commit -m "do something"
        # Merge in the branch mybranch
        git checkout main
        git merge mybranch
    }
}
